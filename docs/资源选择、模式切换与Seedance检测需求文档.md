# 资源选择、模式切换与 Seedance 检测需求文档

## 一、文档目的

本文档用于统一梳理对话侧边栏中与以下能力相关的整体需求：

- 资源选择弹框
- 图片 / 视频 / Agent 模式切换
- 视频模式下文生视频 / 首尾帧 / 全能参考切换
- 图片与视频模型偏好
- Seedance 资源检测、过滤、提示与可用性控制

文档目标是将多轮交互需求收敛为一份可供产品、开发、测试共同使用的正式说明，避免后续口径分散。

## 二、页面定位

- 页面形态：对话侧边栏生成区
- 主类型：AI 功能与提示词类
- 子类型：上传导入类 / 多步骤流程类 / 资源选择类
- 置信度：高

## 三、文档依据

本文档基于以下资料整理：

- 页面实现：`src/components/Sidebar.tsx`
- 资源库实现：`src/components/ResourceLibraryDialog.tsx`
- 用户多轮交互确认与截图反馈

资料冲突时，按以下优先级处理：

1. 用户本次明确确认的需求
2. 当前页面实际实现
3. 历史口头描述与截图参考

## 七、总体目标

### 7.1 资源选择目标

- 用户可在一个统一弹框中完成素材浏览、筛选、检测、勾选、确认添加。
- 资源选择需同时覆盖图片、视频、音频、模特四类资源。
- 在视频参考场景下，资源库需支持单槽位选择；在普通素材添加场景下，支持多选。

### 7.2 模式切换目标

- 图片模式、视频模式、Agent 模式的模型偏好体验需要分层明确。
- 视频模式下需要支持文生视频、首尾帧、全能参考三种输入方式。
- 不同模式的上传入口、模型入口、资源限制需要联动变化。

### 7.3 Seedance 控制目标

- 当视频模型为 Seedance 系列时，涉及图片 / 模特资源的引入必须受 Seedance 检测约束。
- 检测未通过的资源不可进入已选择，也不可继续作为可用素材参与生成。
- 用户需要明确感知哪些资源已通过 Seedance 检测，哪些资源尚未检测，哪些资源不可用。

## 八、核心对象定义

### 8.1 模式定义

| 模式 | 说明 |
|---|---|
| Agent | 通用对话与生成入口，可选择图片模型或视频模型 |
| 图像 | 专注图片生成，仅显示图片模型与图片参数 |
| 视频 | 专注视频生成，仅显示视频模型与视频参数 |

### 8.2 视频输入模式定义

| 输入模式 | 说明 |
|---|---|
| 文生视频 | 仅文本描述生成视频 |
| 首尾帧 | 需要上传或选择首帧图、尾帧图 |
| 全能参考 | 可上传或选择图片、视频、音频等参考素材 |

### 8.3 资源类型定义

| 类型 | 说明 |
|---|---|
| 图像 | 普通图片素材 |
| 视频 | 视频素材 |
| 音频 | 音频素材 |
| 模特 | 模特 / 角色类图片素材 |

### 8.4 Seedance 状态定义

| 状态 | 说明 |
|---|---|
| `untested` | 未经过 Seedance 检测 |
| `pending` | 检测中 |
| `passed` | 已通过 Seedance 检测 |
| `failed` | 未通过 Seedance 检测 |

## 九、模块需求

### 9.1 资源选择弹框 [5]

#### 9.1.1 弹框基础结构

资源选择必须为独立弹框，不再拆成页面侧栏。

弹框结构固定为：

1. 标题区
2. 分类切换区
3. 可展开的搜索入口
4. 文件夹区域
5. 资源列表区域
6. 已选择区域
7. 确认按钮区域

#### 9.1.2 标题区

- 标题：`资源库选择`
- 副标题：说明当前可从资源库选择图像、视频、音频、模特素材添加到当前对话
- 右上角关闭按钮使用叉号图标，不使用上传图标

#### 9.1.3 分类切换区

分类固定为：

- 图像
- 视频
- 音频
- 模特

要求：

- 分类行在资源内容滚动时吸顶
- 分类切换后仅更新对应资源列表，不关闭弹框

#### 9.1.4 搜索与顶部工具区

- 搜索入口默认折叠为小图标
- 搜索图标显示在文件夹图标左侧
- 点击搜索图标后展开搜索输入框
- 搜索按当前 tab 的资源名称过滤

工具区包含：

- 搜索
- 文件夹
- 上传

其中：

- 视频 tab 下的通用上传按钮可按具体业务场景隐藏或迁移
- 图片 / 音频 / 模特 tab 下保留上传入口

#### 9.1.5 文件夹区域

- 文件夹区域一行显示 5 个
- 默认只展示部分文件夹，可通过“展开 / 收起”控制查看更多
- 文件夹卡片仅承担浏览分组展示作用，当前前端实现未接真实切换逻辑

#### 9.1.6 资源列表区域

- 资源区域一行显示 5 个
- 支持多选
- 资源卡支持明显选中态
- 已选择资源在资源列表中需有明显高亮反馈

不同 tab 资源表现：

- 图像 / 模特：展示图片缩略图
- 视频：展示视频封面与播放感知
- 音频：展示音频卡片与波形感知

#### 9.1.7 已选择区域

- 位于弹框底部左侧
- 仅显示已选择内容的类型标签与示意图
- 不显示资源名称
- 卡片尺寸小于资源列表卡片
- 每张卡片支持删除

#### 9.1.8 确认按钮

- 位于底部右侧
- 文案格式：`确定添加（X）` 或 `确定选择（X）`
- `X` 为当前已选择数量
- 当没有已选择内容时按钮禁用

#### 9.1.9 单选与多选规则

| 场景 | 选择模式 |
|---|---|
| 普通素材添加 | 多选 |
| 视频参考槽位 | 单选 |

单选场景中：

- 点击确认后，只将当前槽位对应资源回填
- 不作为普通多素材批量添加

### 9.2 资源添加入口

#### 9.2.1 普通添加入口

对话框底部普通素材添加入口：

- 点击后弹出菜单
- 菜单包含：`本地上传`、`资源库选择`

#### 9.2.2 视频参考入口

视频模式下，当输入模式不是文生视频时，每个参考块点击后展开局部菜单：

- 本地上传
- 资源库选择

要求：

- 点击整块触发展开，不常驻显示操作按钮
- 不显示额外下拉箭头
- 块级标题可省略，块内主文案即类型名称

#### 9.2.3 视频模式下参考块规则

1. 首尾帧模式

- 显示两个块：首帧图、尾帧图
- 两块尺寸需与全能参考的参考块保持一致
- 两块之间显示“与”

2. 全能参考模式

- 显示三个块：图片、视频、音频
- 每个块均支持本地上传与资源库选择

### 9.3 视频模式切换 [3]

#### 9.3.1 模式切换

视频模式下，在“视频”按钮后需有一个下拉模式切换：

- 文生视频
- 首尾帧
- 全能参考

默认值为：`文生视频`

#### 9.3.2 文生视频

- 不显示参考上传区
- 仅基于文本输入生成视频

#### 9.3.3 首尾帧

- 显示首帧图与尾帧图参考区
- 若当前视频模型属于 Seedance 系列，显示模特/角色素材引导条

#### 9.3.4 全能参考

- 显示图片 / 视频 / 音频参考区
- 若当前视频模型属于 Seedance 系列，显示模特/角色素材引导条

### 9.4 模型偏好 [6]

#### 9.4.1 Agent 模式

Agent 模式下，模型偏好保留原始“模型切换”职责，不承担参数设置：

- 支持 `生图片 / 生视频` 两个 tab
- `生图片` tab 内仅显示图片模型列表
- `生视频` tab 内仅显示视频模型列表
- 支持多选模型
- 允许在弹框内连续切换 tab、连续勾选多个模型而不自动关闭弹框
- 只有点击弹框外部区域时才关闭

#### 9.4.2 图片模式

图片模式下，模型偏好弹层结构与视频模式一致，但仅显示图片模型：

- 左侧：图片模型列表
- 右侧：当前图片模型支持的比例参数

图片模式下不显示视频模型切换。

#### 9.4.3 视频模式

视频模式下入口文案调整为：`模型参数偏好`

结构固定为：

- 左侧：视频模型
- 右侧：参数设置

参数需跟随所选模型联动展示当前模型支持的：

- 比例
- 时长
- 分辨率

#### 9.4.4 弹层展示要求

- 模型偏好弹层需完整显示在视口中
- 不得被对话框、侧边栏、其它容器裁切
- 当前实现已改为 portal + fixed 浮层定位

### 9.5 Seedance 检测 [7]

#### 9.5.1 Seedance 检测生效条件

当且仅当满足以下条件时，对图片 / 模特资源启用 Seedance 检测约束：

1. 当前模式为视频模式，且所选视频模型属于 Seedance 系列
2. 或 Agent 模式下选择视频模型时，需要对进入视频参考链路的相关图片素材做同口径限制

当前前端实现中，重点落地在视频模式下的资源选择和本地上传链路。

#### 9.5.2 检测触发时机

#### 资源库中点击资源添加时

- 若资源状态为 `untested`
- 且当前属于 Seedance 检测模式
- 则点击资源时先触发检测
- 通过后才可加入“已选择”
- 未通过则不可加入“已选择”

#### 本地上传图片时

- 当目标链路受 Seedance 约束
- 且本地上传内容为图片
- 则本地上传后先走检测
- 通过才可进入输入框 / 参考位
- 未通过直接拦截并提示错误 toast

#### 手动提交检测时

资源卡右上角 `...` 菜单中提供：

- `提交Seedance检测`

用于对历史资源手动触发检测。

#### 9.5.3 检测通过后的表现

- 资源可正常被选择
- 资源卡左上角显示 Seedance 品牌标识
- 资源在 Seedance 可用过滤中可被筛出

#### 9.5.4 检测失败后的表现

- 不显示“检测未通过”大文案标签
- 资源卡直接置灰
- 不允许继续点击选择
- 若是点击选择时触发失败，不得进入已选择区

提示文案统一为：

`未通过Seedance检测，请更换其他资源`

#### 9.5.5 检测中的表现

- 资源卡显示 `检测中`
- 在检测完成前资源不可再次点击

#### 9.5.6 未检测资源的表现

- 大部分历史资源仍可能处于未检测状态
- 这类资源不显示 Seedance 通过标识
- 视觉上保持普通资源状态
- 仅在真正触发 Seedance 相关使用时才检测

### 9.6 Seedance 标识与过滤

#### 9.6.1 单资源标识

- Seedance 通过资源在左上角显示品牌风格标识
- 不使用普通对号，避免与“已选中”状态混淆
- hover 标识时显示 `Seedance可用`

#### 9.6.2 列表过滤

在图像 / 模特 tab 的资源区域右上方提供：

- `Seedance可用` 过滤勾选项
- 后附一个问号图标

#### 9.6.3 过滤规则

- 勾选后仅显示 `passed` 状态的资源
- 未勾选时显示全部资源，包括未检测资源与已通过资源
- `failed` 状态资源若仍在列表中则保持置灰

#### 9.6.4 说明弹框

点击问号图标后，弹出 `Seedance 主体素材要求` 说明浮层，内容包括：

- 肖像格式、宽高比、尺寸、文件大小
- 音频格式、时长、大小
- 视频格式、分辨率、时长、宽高比、像素、帧率、大小

底部提示：

`满足以上要求并不保证一定通过审核。`

### 9.7 Seedance 引导条

当视频模型属于 Seedance 系列，且当前为 `首尾帧` 或 `全能参考` 模式时，在参考上传区上方显示引导条：

- 文案：`模特/角色素材通过资源库审核通过后方可使用`
- 右侧上传按钮点击后打开资源库弹框
- 打开后默认定位到 `模特` tab

该引导条仅用于说明模特/角色素材使用门槛，不替代普通参考块的上传能力。

| Seedance 自动检测 | 在受限链路点击未检测图片/模特资源 | 先检测，通过后加入，失败则拦截 |
| Seedance 手动检测 | 点击资源卡右上角更多菜单 | 显示提交检测入口并更新状态 |

## 十一、异常与边界处理

| 异常 / 边界 | 处理方式 |
|---|---|
| 资源未检测但当前不受 Seedance 约束 | 允许按普通资源使用 |
| 资源未通过 Seedance 检测 | 置灰，不可选，不进入已选择区 |
| 检测中资源被再次点击 | 不响应二次选择 |
| 本地上传图片在 Seedance 场景检测失败 | 不进入输入框 / 参考位，直接 toast 提示 |
| 资源库无已选择内容 | 确认按钮禁用 |
| 视频 tab 下通用上传入口 | 可按当前业务迁移至参考块或顶部专属入口 |
| 模型偏好弹框位于裁切容器内 | 改用 portal + fixed 避免被裁切 |
| Agent 模式切换 tab 或勾选模型 | 不自动关闭弹框，仅点击外部时关闭 |
| Agent 模式多选全部取消 | 至少保留 1 个模型，避免全空 |

## 十二、当前实现核对结论

## 十三、验收标准

1. 点击普通添加入口时，可选择本地上传或资源库选择。
2. 资源库弹框中必须包含图像、视频、音频、模特四类 tab。
3. 文件夹与资源列表都需一行展示 5 个。
4. 资源库支持多选，并在底部已选择区汇总展示。
5. 视频模式必须支持文生视频、首尾帧、全能参考三种模式切换。
6. 首尾帧与全能参考模式下，参考块点击后可选择本地上传或资源库选择。
7. 视频模型为 Seedance 系列时，图片 / 模特资源在进入可用链路前必须经过检测。
8. 检测通过资源才可加入已选择或输入区。
9. 检测失败资源不得加入已选择，并需要提示统一错误文案。
10. Seedance 通过资源需显示品牌标识，不可与选中态混淆。
11. 图片模式模型偏好只显示图片模型与比例参数。
12. 视频模式模型偏好只显示视频模型与视频参数。
13. Agent 模式模型偏好支持生图片 / 生视频切换，且支持多选模型。
14. Agent 模式下切换 tab 和勾选模型时弹框不得自动关闭。
15. 点击弹框外部区域时，模型偏好弹框应关闭。
16. 图片与视频模式的模型偏好弹层不得被页面裁切。
